This folder is used for all images and icons for the MCG-LINEA site.

Place your final images here with the following file names (or update the HTML/CSS paths accordingly):

Hero / content images:
- warehouse-hero.jpg       -> main hero background on index.html
- parts-closeup.jpg        -> services section image (index.html)
- logistics.jpg            -> services section image (index.html)
- warehouse-detail.jpg     -> services section image (index.html)
- office-detail.jpg        -> hero image on company.html
- bank-building.jpg        -> hero image on payments.html

Branding / icons (optional, recommended for production):
- logo.png or logo.svg     -> company logo (update HTML if you want to use it)
- favicon.png              -> favicon (16x16 or 32x32)
- apple-touch-icon.png     -> icon for mobile shortcuts (e.g. 180x180)
- og-image.png             -> Open Graph preview image (e.g. 1200x630)

You can replace these names with your own; if you do, update:
- index.html
- company.html
- payments.html
- styles.css (for the hero background image on the landing page)
